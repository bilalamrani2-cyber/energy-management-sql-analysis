# Energy Management SQL Analysis

This project is a beginner SQL database analysis project focused on energy production data.

## Project Overview
The database is designed to manage:
- Regions
- Energy types
- Power plants
- Daily production records

The project includes SQL tasks such as:
- Creating tables with primary and foreign keys
- Inserting sample data
- Filtering and sorting records
- Using aggregate functions
- Grouping results
- Writing JOIN queries
- Analyzing production by region and energy type

## SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- COUNT
- AVG
- MIN / MAX
- GROUP BY
- HAVING
- INNER JOIN
- LEFT JOIN
- ALTER TABLE

## Example Analysis Questions
- Which plants have no production records?
- What is the average production by energy type?
- Which region has the highest total production?
- Which days had production above 1500?

## Tools
- MySQL
- SQL

## Purpose
This project was created to practice SQL database design and data analysis concepts for entry-level data analyst roles.
